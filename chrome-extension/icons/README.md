
# Extension Icons

Platzieren Sie folgende Icon-Dateien in diesem Ordner:

- icon16.png (16x16 px)
- icon48.png (48x48 px) 
- icon128.png (128x128 px)

Die Icons sollten das SEO-Analyzer-Logo oder ein Suchlupen-Symbol zeigen.

Für die Entwicklung können Sie auch einfache PNG-Dateien mit entsprechenden Größen erstellen.

## Schnelle Icon-Erstellung:

1. Gehen Sie zu https://www.favicon-generator.org/
2. Laden Sie ein Logo/Symbol hoch
3. Laden Sie die generierten Icons herunter
4. Benennen Sie sie in icon16.png, icon48.png, icon128.png um
